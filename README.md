# bmo17-cprf
Study and implementation of constrained PRFs: construction of BMO17, CJ25 attack, hashed CPRF correction and security analysis

---

### Compile

```bash
make
```

---

### Run

Use the launcher script:

```bash
./start.sh -n|-h|-l <size>
```

| Option | Description |
|------|------------|
| `-n` | Normal CPRF |
| `-h` | Hashed CPRF |
| `-l` | Lazy sampling |
| `<size>` | Number of tests |
